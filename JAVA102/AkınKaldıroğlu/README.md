# Java ile Nesne-Merkezli ve Fonksiyonel Programlama

Bu klasör, **Akın Kaldıroğlu** tarafından Udemy üzerinde verilen **"Java ile Nesne-Merkezli ve Fonksiyonel Programlama"** kursu kapsamında tamamladığım bölüm sonu ödevlerini ve uygulama projelerini içermektedir.

## Kurs Hakkında
Kurs, Java dilinin temellerinden başlayarak ileri seviye nesne tabanlı programlama (OOP) prensiplerini, fonksiyonel programlama yaklaşımlarını ve modern Java özelliklerini derinlemesine ele almaktadır.

## İçerik Yapısı
Her klasör, kursun ilgili bölümüne ait bir ödevi veya uygulama çalışmasını temsil eder.

## Kullanılan Teknolojiler
* **Dil:** Java
* **IDE:** IntelliJ IDEA
* **Versiyon Kontrol:** Git
